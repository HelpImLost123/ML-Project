blur-face-app/
├─ frontend/            # ส่วนของ UI (React หรือ Next.js)
│  ├─ public/          # เก็บไฟล์รูปภาพหรือไฟล์ static เช่นโลโก้ หรือไอคอน
│  ├─ src/             # โค้ดหลักของ React
│  │   ├─ components/  # คอมโพเนนต์ที่ใช้ซ้ำได้ในหลายหน้าของแอป
│  │   ├─ pages/       # หน้าต่าง ๆ จัดการหน้าเว็บ เช่น หน้า Home หรือหน้า Upload
│  │   ├─ App.jsx      # คอมโพเนนต์หลัก คอมโพเนนต์หลักที่รวมทุกหน้าเข้าด้วยกัน
│  │   └─ main.jsx     # จุดเริ่มต้นของแอป ไฟล์ที่สั่งให้แอปทำงานบนหน้าเว็บ
│  └─ package.json     # รายชื่อ dependencies ของโปรเจค
└─ backend/            # ส่วนของ API และการประมวลผล
   ├─ app.py           # Flask API หรือ FastAPI
   ├─ library.txt # ไลบรารีที่ใช้ (เช่น opencv, fastapi, pillow)
   └─ models/          # โค้ดโมเดล AI สำหรับตรวจจับใบหน้า
